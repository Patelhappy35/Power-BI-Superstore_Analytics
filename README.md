### Sales Performance Dashboard

## Project Overview

This Power BI project provides insights into sales performance using the Superstore dataset. The dashboard helps analyze revenue, profit, quantity, and discount across different categories and regions.

## Tools Used

* Power BI Desktop
* Power Query Editor
* Microsoft Excel

## Dataset

* Dataset Name: Sample Superstore Sales Dataset
* Source: Kaggle

## Features

* Data Cleaning and Transformation
* 4 KPI Cards
* Sales by Category Bar Chart
* Profit by Region Bar Chart
* Region and Category Slicers
* Page-Level and Visual-Level Filters
* Visual Interactions
* Professional Theme and Formatting
* Two Report Pages

## Steps Performed

1. Connected Excel dataset to Power BI.
2. Cleaned and transformed data using Power Query.
3. Created KPI Cards for Sales, Profit, Quantity, and Average Discount.
4. Created Sales by Category and Profit by Region bar charts.
5. Added Region and Category slicers.
6. Applied page-level and visual-level filters.
7. Formatted the report using a professional theme.
8. Created a second page with detailed order data.

## Report Pages

### Page 1: Sales Performance Dashboard

Displays KPI cards, charts, slicers, and filters for interactive analysis.

### Page 2: Detailed Order Data

Contains a table visual showing detailed order information.
